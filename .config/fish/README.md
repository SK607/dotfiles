# Fish plugins

- [fisher](https://github.com/jorgebucaran/fisher) - package manager
- [autopair](https://github.com/jorgebucaran/autopair.fish) - autocomplete brackets/quotes
- [fzf.fish](https://github.com/PatrickF1/fzf.fish) - extended fish key bindings
- [puffer-fish](https://github.com/nickeb96/puffer-fish) - text expansion (..../!!/!$)

More plugins at [awsm.fish ](https://github.com/jorgebucaran/awsm.fish)


# Installation

```fish
curl -sL https://git.io/fisher | source && fisher install jorgebucaran/fisher
fisher install jorgebucaran/autopair.fish 
fisher install jPatrickF1/fzf.fish
fisher install nickeb96/puffer-fish
```
